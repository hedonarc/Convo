# Backend API

API usage and endpoint references for backend services.

## Access

- Admin: `http://127.0.0.1:8000/admin/`
- Profiling: `http://127.0.0.1:8000/silk/`

## Auth Endpoints

- `POST /api/register/`
- `POST /api/login/`

Add endpoint groups here as the API grows (users, conversations, messages).
